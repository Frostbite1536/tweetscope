# Contributing to Tweetscope

Thank you for considering contributing to Tweetscope! We welcome contributions from the community.

## Development Guide

For detailed information on setting up the development environment and the structure of the project, see the [Development Guide](DEVELOPMENT.md).

## Reporting Issues

Found a bug or have a feature request? Open an issue on [GitHub](https://github.com/maskys/tweetscope/issues).

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
